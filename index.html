View project on GitHub

About

This project is used to get the movies data freely stored in The Numbers. The intention is to easily gather the data for research purpose.

Features

The main feature is that this can be used to gather the movies data. The data will be stored in SQL Server. You can change that by the way since I only use DBML to manage my data connection.

Added feature is that you can export the data to Excel so you can easily print it. If you prefer, you could export it from your DBMS for faster and more precise result with custom queries.
